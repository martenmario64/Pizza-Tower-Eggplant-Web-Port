# Pizza Tower Eggplant Build - Web Port

A custom version of Pizza Tower's Eggplant Build meant

for your own web browser, now for multiple reasons,

You will have to use the releases guide to localhost it,

as You will require your own copy of Pizza Tower's

Eggplant Build, the main reason why I cannot host it is

there is no easy way to do it currently, I know it's

a hassle, but it's so this repo does not get DMCA'd,

no I will not tell You where to get a copy of this Beta

Build of Pizza Tower.
